# English version:

## Details:

- Minecraft version: `Fabric 1.19.2`
- Gui keybind - **``Delete``** (Attention! - In Panic Mode GUI won't open, the hack is entirely disabled)
- Hack version - **`v1.2.0`**
- List of modules - [(Click)](#list-of-modules)
- Panic mode activation - Hold down Ctrl + Delete for 5 seconds, follow the instructions appeared on the screen (Panic mode can be disabled only by restarting the game)

## Requirements

- [FabricApi 1.19.2](https://www.curseforge.com/minecraft/mc-mods/fabric-api/files/4902659)
- [Java 17+](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

## List of modules:

- **Combat**
    - Elytra Hunter
    - Trigger Bot
- **Visual**
    - Truesight
    - Client time
- **Player**
    - Gui Move
- **World**
    - Chest Stealer
    - Chorus farmer
- **Miscellaneous**
    - Auto sell
    - Player searcher

# Русская версия:

## Подробности:

- Версия Minecraft: `Fabric 1.19.2`
- Кейбинд GUI - **``Delete``** (Внимание! - В режиме паники GUI не откроется, чит полностью отключён)
- Версия чита - **`v1.2.0`**
- Список модулей - [(Click)](#список-модулей)
- Активация режима паники - Зажмите Ctrl + Delete на 5 секунд и следуйте инструкции на экране (Режим паники может быть отключён только перезапуском игры)

## Требования

- [FabricApi 1.19.2](https://www.curseforge.com/minecraft/mc-mods/fabric-api/files/4902659)
- [Java 17+](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

## Список модулей:

- **Combat**
    - Elytra Hunter (Ищет игроков с элитрами и аимится на них)
    - Trigger Bot
- **Visual**
    - Truesight
    - Client time
- **Player**
    - Gui Move
- **World**
    - Chest Stealer
    - Chorus farmer (Автоматически целится и стреляет в цветки хоруса)
- **Miscellaneous**
    - Auto sell
    - Player searcher (Поиск игрока по нику, если среди игроков которых вы можете видеть есть данный игрок то вам выведутся координаты игрока, при нажатии на стрелке вверх ваша камера направиться на него)
